# Analog Audio Effect Processor

## Overview

Fully analog audio processing unit incorporating multiple audio effects, equalization stages, and dedicated power supply circuitry for audio signal enhancement.

## Features

* Equalizer section
* Tremolo effect
* Phaser effect
* Overdrive effect
* Analog signal processing
* Dedicated regulated power supply

## Technologies

* Analog Electronics
* LTSpice
* Altium Designer
* SolidWorks
